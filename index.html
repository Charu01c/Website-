🚀 How to Create Your First Website on GitHub – Step by Step Guide

✅ Step 1: Create a GitHub Account

Visit https://github.com and sign up for a free account by filling in your username, email, and password. Verify your email address after signup.

✅ Step 2: Create a New Repository

  1. Log in to GitHub.
  2. Click the ➕ icon (top right corner) → Select New Repository.
  3. Name the repository, e.g., My-First-Website.
  4. Set repository to Public.
  5. Check Initialize this repository with a README.
  6. Click Create Repository.

✅ Step 3: Add Your Website Files

1. Open your repository.
2. Click Add file → Upload files.
3. Upload files: index.html, style.css, script.js, images.

Tip: The main page should be index.html.

✅ Step 4: Enable GitHub Pages

  1. Go to Settings tab.
  2. Scroll to Pages section.
  3. Select main branch under Source and click Save.
  4. Get URL like:
    https://yourusername.github.io/repository-name/

✅ Step 5: View Your Live Website

Visit the generated URL and your website will be live on the web! 🌐

✅ Pro Tips

  • Use folders: /css, /js, /images.
  • Keep index.html simple and clean.
  • Commit descriptive messages when updating files.
  • Use GitHub version control features for better tracking.
